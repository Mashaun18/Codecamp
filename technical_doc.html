Introduction

CSS transitions allow you to create smooth animations when properties of an element change.

They are used to provide visual feedback or enhance the user experience on a webpage.

Transition properties allow elements to change values over a specified duration, animating the property changes, rather than having them occur immediately.

So, if we have, say, a box with a red background that we want to change to a green background when it is hovered, we can reach right for the transition property to move between background colors:

Transition Properties

You can specify a particular property as we have above, or use a value of all to refer to transition properties

You can transition various CSS properties, such as:

  • Width: For smoothly resizing an element.
  • Height: For smooth changes in height.
  • Color: To transition text or background color changes.
  • Opacity: To create smooth fading effects.
  • Transform: For animations like scaling, rotating and translating.

For example, to create a color transition, you can use:

transition: color 0.5s ease-in-out;
Timing Functions

Timing functions control the speed of transitions. Common timing functions include:

  • ease
  • ease-in
  • ease-out
  • ease-in-out
  • linear

For example, to create a smooth in-out transition, you can use:

transition: width 1s ease-in-out;

Transition Transformation

When used together, these properties allow you to create simple animations and add valuable interaction and visual feedback for your users.

So what are transforms and transitions? At their most basic level, transforms move or change the appearance of an element, while transitions make the element smoothly and gradually change from one state to another.

transition: transform 500ms 5s;
Transformation

CSS transforms-how to make an element change from one state to another. With the CSS transform property you can rotate, move, skew, and scale elements.

Transforms are triggered when an element changes states, such as on mouse-hover or mouse-click. The methods of transformation include:

  • scale
  • rotate
  • translate
  • skew
  • transform-origin
